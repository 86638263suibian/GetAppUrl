# GetAppUrl

## Usage ##
  [*] Useage: java -jar GetAPPurl {Options} {target App}
  [*] Options:                                          
  [*]         -A: 获取dex，arsc，so中全部字符串                                                
  [*]         -au: 获取dex，arsc中Url                 
  [*]         -du: 获取dex中的全部url
